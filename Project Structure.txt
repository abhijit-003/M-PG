               │
               ▼

               ▼

               │
               ▼
┌──────────────────────────────┐
│         Routes Layer         │

└──────────────┬───────────────┘
               │
               ▼
┌──────────────────────────────┐
│        Middleware Layer      │
│  - Auth (JWT যাচ)           │
│  - Role Check (Admin?)       │
│  - Validation                │
└──────────────┬───────────────┘
               │ Valid Request
               ▼
┌──────────────────────────────┐
│       Controller Layer       │
│   (Handles req & res)        │
└──────────────┬───────────────┘
               │
               ▼
┌──────────────────────────────┐
│        Service Layer         │
│   (Business Logic)           │
│   - Room capacity check      │
│   - Rent calculation         │
└──────────────┬───────────────┘
               │
               ▼
┌──────────────────────────────┐
│         Model Layer          │
│   (SQL Queries)              │
└──────────────┬───────────────┘
               │
               ▼
┌──────────────────────────────┐
│        MySQL Database        │

└──────────────┬───────────────┘
               │ Query Result
               ▲
               │
┌──────────────┴───────────────┐

└──────────────┬───────────────┘
               │
               ▼
┌──────────────────────────────┐

               │
               ▼
┌──────────────────────────────┐
│     Controller Response      │
│     (JSON to frontend)       │

               │
               ▼
